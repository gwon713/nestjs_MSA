## Installation

```bash
$ npm install
```

## Running the app local

```bash
# gateway
$ nest start gateway
# watch mode
$ nest start gateway --watch

# auth
$ nest start auth
# watch mode
$ nest start auth --watch
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
